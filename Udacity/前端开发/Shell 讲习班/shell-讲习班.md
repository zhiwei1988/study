**在执行 shell 命令时，不清楚该用`"`还是`'`时，优先使用`'`**

### 起始文件

在 Windows 或 Mac 上是 `.bash_profile`

在 Linux 上 `.bash_profile` 只对非交互式 shell 起作用，交互式 shell 使用 `.bashrc`

### curl

`curl -o dictionary.txt -L 'http://t.cn/RYkeaZi'`

> `-o` 保存到指定文件
>
> `-L` 放置页面重定向

### less

使用 `Space` 或 `↓` 进行向下翻页；使用 `b` 进行向上翻页；使用 `/` 进行查找

### grep

> `-c` 统计匹配的行数
